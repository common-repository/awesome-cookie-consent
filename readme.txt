== Description ==

Awesome way to setup GDPR Cookie Consent Banner and customize with live preview to match your Cookie Compliance Consent requirements and website layout.

= Key Features include: =

* Fully Customizable with live preview (like Color, Position, Message and more)
* Redirect to specified page for more cookie information
* Responsive design
* Multiple choice for position
* Option to show/hide cookies message
* Put the cookie bar in either the header or the footer
* Bootstrap 3 and Bootstrap 4 support
* Multilingual compatible with polylang plugin
* Multilingual compatible with WPML plugin

== Installation ==

1. Install Awesome GDPR Compliant Cookie Consent and Notice either via the WordPress.org plugin directory, or by uploading the zip file to your server.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. That’s it!  Now you can configure the plugin settings.

== Frequently Asked Questions ==

= Can I customize this plugin? =
Yes, we offer many customization options, from the position of your popup, backgroun color of the popup, buttons, and text.

= I don’t see the banner in my site =
Try this way:
  1. Deactivate all plugin and try
  2. If it works activate one pluigin at time and try
  3. If it doesn’t work activate standard theme and try

= The cookie notice popup doesn’t work on my browser? =
Please report a bug on the support forum. Be sure to include the following information:
* Your Website URL
* WordPress version e.g. 4.2.2 (found in bottom right hand corner of the dashboard)
* Browser e.g. Safari, FireFox, Chrome, IE
* Details of the problem

= What does this plugin do? =
* Add a cookie popup to the top/bottom/left/right of all pages on your website, notifying the visitor that you have a clearly defined privacy and cookie policy.
